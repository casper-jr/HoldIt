# 장기투자 우량주 평가 가이드라인

본 가이드라인은 저평가된 배당 성향의 우량주를 발굴하고 장기 투자 적합성을 평가하기 위한 점수 산정 기준을 명시합니다.
총점은 100점 만점이며, Buffett & Lynch의 투자 철학을 반영하여 수익 창출력, 성장성 및 재무 안전성, 주주환원 정책의 3가지 카테고리로 나뉘어 평가됩니다.

> **개편 배경**: 기존 시스템의 핵심 문제였던 Value Trap(저PER이지만 성장이 없는 기업) 미필터링을 보완하기 위해,
> 버핏이 강조하는 ROE·FCF와 린치의 PEG를 도입하고 PER 단독 비중을 낮췄습니다.

---

기업이 주주 자본을 얼마나 효율적으로 활용하는지, 그리고 현재 주가 대비 얼마나 저평가되어 있는지를 평가합니다.

## 2. 성장성 및 재무 안전성 (총 30점)

기업의 성장 속도 대비 주가의 합리성, 장기 생존을 위한 재무 건전성, 지속적인 경쟁 우위를 평가합니다.

| 평가 항목 | 세부 기준 | 부여 점수 | 비고 |
| --- | --- | --- | --- |
| **경제적 해자 및 성장성** (정성적 판단, 최대 10점) | 강력한 해자 | 10점 | 가격 결정력, 시장 점유율 유지 능력, 브랜드 등을 통합 평가 |
| | 보통 해자 | 5점 | 기존 '미래 성장 잠재력'과 '세계적 브랜드' 항목을 통합 |
| | 해자 없음 | 0점 | |
| **PEG** 주가수익성장비율 (최대 10점) | 0.5 미만 | 10점 | 린치: "PEG 0.5 이하는 최고의 기회" |
| | 1.0 미만 | 7점 | 린치: "PEG 1.0 이하는 합리적 가치" |
| | 1.5 미만 | 4점 | PEG = PER ÷ EPS 연간 성장률(%) |
| | 2.0 미만 | 2점 | EPS 성장률 데이터 없거나 성장률 ≤ 0이면 0점 처리 |
| | 2.0 이상 | 0점 | |
| **부채비율** 재무 건전성 (최대 10점) | 30% 미만 | 10점 | 린치: "부채 없는 회사는 망하지 않는다" |
| | 60% 미만 | 7점 | 부채비율 = 총부채 / 자기자본 × 100 |
| | 100% 미만 | 4점 | |
| | 200% 미만 | 2점 | |
| | 200% 이상 | 0점 | |

> **경영진 역량 (정성)**: 버핏·린치 모두 경영진의 도덕성과 자본 배분 능력을 중시하나, 정량화 불가하여 총점 합산에~~서 제외하고
> `qualitative_assessments` 테이블에 별도 기록·관리합니다. (EXCELLENT, PROFESSIONAL, POOR)

## 3. 주주환원 정책 (총 30점)

기업이 벌어들인 이익을 주주에게 얼마나 지속적이고 적극적으로 환원하는지를 평가합니다.

| 평가 항목 | 세부 기준 | 부여 점수 | 비고 |
| --- | --- | --- | --- |
| **배당수익률** (최대 10점) | 7% 초과 | 10점 | |
| | 5% 초과 | 7점 | |
| | 3% 초과 | 5점 | |
| | 3% 미만 | 2점 | |
| **배당 성장성** (최대 10점) | 10년 이상 연속 인상 | 10점 | 버핏: 배당의 총량보다 지속적 우상향이 기업 자신감의 증거 |
| | 7년 이상 | 8점 | KR: DART alotMatter (최대 12년치) |
| | 5년 이상 | 6점 | US: yfinance 연도별 배당금 분석 |
| | 3년 이상 | 3점 | |
| | 해당 없음 | 0점 | |
| **자사주 소각 실적** (최대 10점) | 최근 1년 내 소각 공시 확인 | 10점 | 단순 매입 보유가 아닌, 실제 주식 수를 줄여 주당 가치를 높이는 소각 실적만 평가 |

## 4. 최종 투자 점수 및 등급 판정

위 3가지 카테고리의 점수를 합산(총 100점 만점)하여 최종 투자 등급을 결정합니다.

| 투자 점수 | 투자 등급 | 설명 및 대응 전략 |
| --- | --- | --- |
| **80점 초과** | **A** | **장기투자 적합, 적극 매수** |
| **70 ~ 80점** | **B** | **장기투자 적합, 매수 고려** |
| **50 ~ 70점 미만** | **C** | **이미 가지고 있다면 장기투자 유지 (홀딩), 신규로는 추천하지 않음** |
